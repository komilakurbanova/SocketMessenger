# DBManager

## Необходимый список команд

`/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"`

`brew install cmake`

`brew install postgresql`

`brew install libpqxx`

## Запуск local версии

`cmake .`

`make`

`cd bin && ./db_manager`
